# Contributing Guidelines

Thank you for considering contributing!

## How to Contribute
1. Fork the repo
2. Create a new branch: `feature/my-feature`
3. Make your changes
4. Submit a Pull Request

## Code Style
- Follow PEP8 for Python.
- Add comments for clarity.
- Keep functions modular.

## Reporting Issues
Open an Issue with:
- Expected result
- Actual result
- Steps to reproduce
